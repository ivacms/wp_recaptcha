В плагине отсутствуют две опции для ввода секртеного ключа, и ключа сайта. Данные ключи нужно указать внутри самого класса.

private $site_key='';
	
private $secret_key='';

Данный плагин был написан специально для сайта http://ivacms.ru, с целью показать как именно разрабатываются плагины для Wordpress, и как довольно просто на программно уровне подключить каптчу (2 версия) от компании Google.
