В плагине отсутствуют две опции для ввода секретного ключа, и ключа сайта. Данные ключи нужно указать внутри самого класса.

private $site_key='';
	
private $secret_key='';

Данный плагин был написан специально для сайта https://ivacms.ru, с целью показать как именно разрабатываются плагины для Wordpress, и как довольно просто на программном уровне подключить каптчу (V2) от компании Google.
